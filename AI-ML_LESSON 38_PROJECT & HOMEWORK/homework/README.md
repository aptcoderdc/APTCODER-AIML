## AI in Healthcare - Patient Monitoring

This homework explores patient monitoring using AI techniques.

## Overview
In this homework, we plot the patient's heart rate over time to monitor their health status.

## Concepts Covered
Data Visualization
Patient Monitoring

## Required Libraries
NumPy
Matplotlib

## Setup
Install the required libraries: pip install numpy matplotlib
Download or clone the project files.

## How to Run
Open the patient_monitoring.py file in your Python environment.
Run the script.

## Example Output
A line plot will be displayed showing the patient's heart rate over time.

## Conclusion
This homework demonstrates a basic method for monitoring patient health using AI techniques, which can help healthcare professionals in assessing and managing patient conditions.
